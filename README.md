# Features

* User Authentication (JWT)
* Create Groups
* Add Members by Email
* Add Shared Expenses
* Equal Expense Splitting
* Per-Group Balance Tracking
* Global Balance Summary
* Settlement Recording
* Settlement History
* Mobile Responsive UI

---

# Tech Stack

## Frontend

* React
* Vite
* Tailwind CSS
* Axios

* Node.js
* Express.js
* MongoDB Atlas
